Help & Usage Guide

Advanced Serial Monitor — Web Serial API Edition

Requirements

Works in Chrome, Edge, or Opera (desktop, v89+). Requires a USB-to-serial adapter or a device with native USB serial (Arduino, ESP32, STM32…).

Connecting — Port row

  • Set Baud Rate, Data bits, Stop bits, and Parity to match your device.
  • Click Connect — the browser shows a port picker dialog.
  • The status pill turns green when the port is open.
  • Click Disconnect to close the port safely.

Display modes — View row

  • RX display — controls how received bytes are shown. Changing it re-renders the entire log instantly.
  • TX display — controls how sent bytes are shown in the log. Independent from RX.
  • Text — decoded UTF-8 string.
  • HEX — space-separated uppercase hex, e.g. 41 42 43.
  • DEC — decimal integers, e.g. 65 66 67.
  • BIN — 8-bit binary, e.g. 01000001 01000010.

Sending data — Send row

  • Select the TX input format (left dropdown) — this controls how you type the data.
  • In HEX: type space-separated hex bytes: DE AD BE EF
  • In DEC: type space-separated decimals: 65 66 67
  • In BIN: type space-separated 8-bit groups: 01000001 01000010
  • Choose a line ending — CR+LF is standard for most Arduino sketches.
  • Press Enter or click Send.

Log colours

RX blue — received data   TX green — sent data   amber italic — system messages

Options

  • Auto-scroll — pins the view to the latest line.
  • Timestamps — shows HH:MM:SS.mmm per entry.
  • Clear — wipes log and resets counters.
  • Export — saves the full log as a .txt file.

Keyboard shortcuts

  • Enter — send data
  • ↑ / ↓ — navigate command history (no consecutive duplicates)
  • Ctrl + L — clear log
  • Ctrl + E — export log
